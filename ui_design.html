Fun Quizzes

Introduction

An application that can read up the contents of a quiz (csv, database table, hard-coded, ...) and display the quiz for a user to play.

The contents of quiz will be represented in a table format such as CSV and might look like the following: 

                    IMAGE_NAME, QUESTION, ANSWER_A, ANSWER_B, ANSWER_C, ANSWER_D, CORRECT_ANSWER
                    balloonfish.jpg, Which fish is this?, Trunkfish, Hogfish, Balloonfish, Chub, C
                    moray.jpg, What the heck is this?, Viper, Tequila, Green Moray, Manytooth Conger, C
The app will include such features as a progress bar, radio button group, star ratings,

User Interface Design and Prototype

Navigation Structure

The home page (index.html) will be the quiz. Upon landing on the home page the quiz will be served up. When the quiz is finished it will have a "Try Again" button which will then restart the quiz.
